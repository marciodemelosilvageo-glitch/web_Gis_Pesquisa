Como usar o GeoCode

🗺️ 1. Navegando no Mapa

O mapa exibe as divisões territoriais. Use o botão de camadas flutuante no canto superior direito para Ligar/Desligar as opções (Municípios, Bairros, Favelas, etc.) e para alterar o fundo do mapa (ex: visão de satélite).

🖱️ 2. Selecionando Áreas

Clique Simples: Clique em um polígono colorido para gerar automaticamente um "Raio-X" populacional e ver os detalhes da tabela inferior.

Seleção Múltipla: Ative o botão "Multiseleção" (ideal para celular) ou segure Ctrl no teclado e clique em várias regiões ao mesmo tempo para agregar e somar os dados instantaneamente.

🔍 3. Buscas Avançadas

Utilize a barra no topo para encontrar municípios, bairros ou comunidades. Dica de Ouro: Se o nome for muito comum, use uma vírgula para especificar o município. Exemplo: digite Cacimba, Teresina para buscar a localidade Cacimba apenas dentro de Teresina.

📥 4. Exportação de Dados

Após selecionar feições ou fazer uma busca, role a página até a tabela de "Especificação das Feições". Lá você encontrará botões para baixar os dados em CSV (planilha do Excel) ou KML (Google Earth).
